We Accept card logos (invoice)
==============================
Add these image files here to use your own card logos (e.g. from your reference screenshot):
  - amex.png       (American Express)
  - mastercard.png
  - visa.png
  - discover.png

Same size/spacing as reference: use square or horizontal logos, similar height (~24–28px display).
If a file is missing, the invoice falls back to CDN logos.
